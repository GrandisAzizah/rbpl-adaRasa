# RBPL-AdaRasa

Sistem Manajemen Pesanan Katering
Aplikasi web untuk mengelola pemesanan katering, mencakup alur kerja antara Admin, Tim Dapur, dan Tim Pengantaran.
Fitur

- Manajemen pesanan, menu, dan data pelanggan
- Notifikasi untuk tim dapur dan pengantaran
- Pembaruan status pesanan
- Laporan pesanan dan menu terlaris
- Multi-role login (Admin, Tim Dapur, Tim Pengantaran)
